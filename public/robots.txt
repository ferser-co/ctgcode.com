# ===========================================================================
# CTG Code — reglas de rastreo
#
# El sitio es público, estático y sin áreas privadas: se permite el rastreo
# completo. Los agentes de IA se declaran UNO A UNO y de forma explícita
# porque varios (Google-Extended, Applebot-Extended) NO heredan el permiso
# de `User-agent: *`: si no aparecen nombrados, se comportan como excluidos
# de las respuestas generativas.
#
# Es una decisión deliberada de visibilidad (GEO/AIO): queremos que los
# asistentes puedan leer y CITAR el sitio como fuente.
# ===========================================================================
